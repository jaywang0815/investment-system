# 投資管理系統 - 設定說明

## 步驟總覽
1. 安裝 Python 套件
2. 設定 Supabase 資料庫
3. 設定 Google OAuth 登入
4. 設定 LINE Notify
5. 匯入 Excel 資料
6. 啟動系統

---

## 步驟 1: 安裝套件

在 Terminal (終端機) 執行:
```bash
cd /Users/jay/Desktop/investment-system
pip3 install -r requirements.txt
```

---

## 步驟 2: 設定 Supabase

1. 前往 https://supabase.com → 建立免費帳號
2. 建立新專案 (名稱: investment-system)
3. 等待專案建立完成後，點選左側 **SQL Editor**
4. 複製 `scripts/schema.sql` 的全部內容貼入並執行
5. 到 **Settings → API** 複製:
   - `Project URL` → 填入 `SUPABASE_URL`
   - `service_role` key → 填入 `SUPABASE_KEY`

---

## 步驟 3: 設定 Google OAuth

1. 前往 https://console.cloud.google.com
2. 建立新專案
3. 搜尋並啟用 **Google+ API** 和 **Google Identity**
4. 左側選單 → **APIs & Services → Credentials**
5. 點選 **+ CREATE CREDENTIALS → OAuth 2.0 Client ID**
6. Application type: **Web application**
7. Authorized redirect URIs 新增:
   - `http://localhost:8501/oauth2callback`
8. 複製 Client ID 和 Client Secret

---

## 步驟 4: 建立設定檔

```bash
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
```

用文字編輯器開啟 `.streamlit/secrets.toml` 填入所有資料:
- Supabase URL 和 Key
- Google Client ID 和 Secret
- 你的 Gmail 地址 (allowed_emails)
- LINE Token (步驟5)

---

## 步驟 5: 設定 LINE Notify

**LINE Notify (每日報告):**
1. 前往 https://notify-bot.line.me/my/
2. 點選 **Generate token**
3. 選擇要接收通知的群組或個人聊天
4. 複製 token 填入 `LINE_NOTIFY_TOKEN`

**LINE Messaging API Bot (可選, 讓客戶查詢):**
1. 前往 https://developers.line.biz
2. 建立 Provider → 建立 Messaging API channel
3. 複製 Channel Secret 和 Channel Access Token
4. Webhook URL 填入 Bot 伺服器網址

---

## 步驟 6: 匯入 Excel 資料

```bash
cd /Users/jay/Desktop/investment-system
python3 scripts/import_excel.py
```

按照提示輸入 Supabase URL 和 Key，系統會自動匯入客戶和 SN 商品資料。

---

## 步驟 7: 啟動系統

```bash
cd /Users/jay/Desktop/investment-system
streamlit run app.py
```

瀏覽器會自動開啟 http://localhost:8501

---

## 步驟 8: 啟動 LINE Bot (可選)

```bash
# 設定環境變數
export LINE_CHANNEL_SECRET="你的Secret"
export LINE_CHANNEL_ACCESS_TOKEN="你的Token"
export SUPABASE_URL="你的URL"
export SUPABASE_KEY="你的Key"

# 啟動 Bot 伺服器
uvicorn line_bot_server:app --port 8080
```

如需讓 LINE 可以連到你的電腦，使用 ngrok:
```bash
brew install ngrok
ngrok http 8080
# 複製 https:// 網址填入 LINE Developer Console 的 Webhook URL
```

## 常見問題

**Q: 為什麼看不到股票現價?**
A: Yahoo Finance 有時會延遲，在美股休市時間價格不會更新。正常情況。

**Q: PDF 中文顯示亂碼?**
A: 確認 Mac 上有 PingFang 字型 (正常 Mac 都有)。

**Q: LINE Bot 沒有回應?**
A: 確認 Webhook URL 已設定且 ngrok 正在執行。
